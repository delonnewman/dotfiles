NAME - Dotfiles Makefile
========================

SYNOPSIS
========
```
             setup - sets things up from scratch (installs homebrew, install basic tools for setup)
        copy-files - copies dotfiles to ~/
        show-files - shows files as they've been read from Manifiest
        copy-fonts - copy fonts from fonts/ to ~/Library/Fonts/
    install-basics - install basic tools
 install-mac-tools - install tools for macOS (see Brewfile)
 upgrade-mac-tools - upgrade tools for macOS (see Brewfile)
         vim-setup - install vim plugins
    set-fish-shell - set fish as the default shell (on POSIX systems)
```
